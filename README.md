# SwingInput

## Usage

With gradle:
```kotlin
repositories {
    mavenCentral()
    maven { url = uri("https://jitpack.io") }
}

dependencies {
    // from the main branch
    implementation("com.github.gabrielemercolino:SwingInput:main-SNAPSHOT")
    // from a specific tag
    implementation("com.github.gabrielemercolino:SwingInput:v0.5.0")
}
```
